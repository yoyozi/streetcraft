## The street vendor helper website

Language: Javascript
Framework: Vanilla JS static (changes in future)
Package management: WebPack

### To build

Must have node on local

```
git clone ...
npm install
npm run start or build
```

The build will build out all to ./dist folder so you can just run the index from there

To dev: run start and then develop using your favorite IDE
